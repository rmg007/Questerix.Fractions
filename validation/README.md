# Validation Materials — Questerix Fractions

This folder holds all materials for Phase 2 (internal playtest, 8–10 students) and Phase 3 (classroom test, 25–30 students). The validation plan is governed by `docs/40-validation/playtest-protocol.md` and `docs/40-validation/learning-hypotheses.md`.

## Structure

```
validation/
  phase-2/
    consent-form.md          Parent consent (≤300 words, opt-in)
    pre-test.md              Paper pre-test: 6 items (K) / 7 items (Grade 1+)
    post-test.md             Parallel-form post-test
    observation-template.md  Per-session observer notes
    analysis-template.md     Paired t-test scaffolding + qualitative coding
    recruitment-letter.md    Outreach to homeschool/classroom partners
  phase-3/
    classroom-protocol.md    25–30 student concurrent observation rotation
    teacher-prep.md          30-minute teacher briefing script
    data-export-checklist.md Device-by-device JSON export procedure
```

## Pre-registration

Per `learning-hypotheses.md §5`, freeze all success thresholds and falsification criteria before collecting data. The git commit locking that file is tagged `validation-v1-prereg`. Do not modify thresholds after tagging.

## Data Storage

All data collected under these protocols is stored locally. No cloud uploads. See `consent-form.md` and `data-export-checklist.md` for the specific privacy rules.
