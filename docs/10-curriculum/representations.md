# Visual Representation Registry

This registry defines the visual models used to represent fractions. Using multiple representations is a core pedagogical requirement (§10).

## Summary

| Representation | Description | Best For | Complexity |
|---|---|---|---|
| **Area Model** | Divided 2D shapes (rectangles, circles) | Parts-of-a-whole; partitioning | 1-4 |
| **Set Model** | Group of discrete objects (stars, cookies) | Parts-of-a-set; sharing | 2-5 |
| **Linear Model** | Fractions on a number line or bar | Magnitude; benchmarks; equivalence | 3-5 |

---

## Area Model
**Description:** A continuous 2D shape partitioned into equal areas.
**Visual Variations:**
- Rectangles (easiest to partition)
- Circles (introduces radial thinking)
- Irregular symmetric shapes (trapezoids, hexagons)
**Usage Rule:** Do not use area models exclusively; ensures concepts generalize beyond "divided shapes."

## Set Model
**Description:** A collection of discrete items where a subset is identified or selected.
**Visual Variations:**
- Uniform objects (all cookies same size)
- Varied objects (different types of fruit)
**Usage Rule:** Introduce in Level 2 as a supporting representation. Primary in Level 4.

## Linear Model (Number Line)
**Description:** Fractions represented as points or segments on a 1D line.
**Visual Variations:**
- Empty number line (landmarks only)
- Divided number line (ticks for every unit)
- Fractional bar (hybrid between area and linear)
**Usage Rule:** Introduce in Level 3 as a supporting representation (single benchmark tick). Primary in Level 8.
