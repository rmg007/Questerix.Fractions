# Cultural Contexts Registry

This registry ensures global neutrality and equity by controlling the contexts (scenarios) used in curriculum prompts.

## Approved Contexts

These contexts are globally recognizable and pedagogically neutral.

| Context | Description | Category |
|---|---|---|
| `null` | Bare imperative ("Split this rectangle.") | None (Baseline) |
| `cookie` | Sharing a round or rectangular snack | Sharing |
| `paper-fold` | Folding a piece of paper | Manipulation |
| `garden-bed` | Dividing a plot of land for planting | Area |
| `ribbon` | Cutting a length of fabric or string | Length |
| `block` | Stacking or dividing toy blocks | Discrete |
| `fruit-slice` | Slicing an apple, orange, or melon | Sharing |
| `water-cup` | Filling a container to a level | Capacity |
| `balloon` | Grouping balloons by color | Set |
| `beads-on-string` | Patterning or counting beads | Set |
| `paper-boat` | Multiple boats on water | Set |
| `leaf` | Parts of a leaf or groups of leaves | Nature |
| `sliced-bread` | Loaf of bread cut into slices | Area/Set |

## Limited Contexts

Used sparingly (≤ 10% of items) as they are common but Western-leaning.

- `pizza`
- `pie`
- `round-cake`

## Banned Contexts

These contexts are prohibited to avoid bias, exclusion, or inappropriate content.

- **Holiday-specific:** Halloween, Thanksgiving, Christmas, etc.
- **Substance-related:** Alcohol, medicine.
- **Highly specific sports:** Baseball (outside US/Japan), Cricket (outside Commonwealth).
- **Currency-specific:** Dollars, Pounds (use generic "coins" or "tokens" instead).
- **Socio-economic markers:** Yachts, mansions, specific luxury brands.
