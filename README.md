# Questerix Fractions

Browser-based math game teaching K–2 fraction concepts via drag-and-tap mechanics. Phaser 4 + TypeScript + Vite + Dexie.

**Status:** Phase 1 build complete. Sprint 0 in-flight (see `PLANS/master-plan-2026-04-26.md`).

---

## Quick start

```bash
npm install
npm run dev:app   # http://localhost:5000
```
