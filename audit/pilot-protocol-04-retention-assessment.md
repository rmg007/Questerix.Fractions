# Pilot Protocol 4: Retention-Assessment Protocol (Round 3)

**Version:** 1.0  
**Date:** 2026-05-05  
**Plan:** Plan 15 (pilot-protocol), Phase 4  
**Scope:** Long-term retention & concept durability, L1–L9 focus; validates mastery persistence  
**Participants:** 8–12 K–2 learners (same cohort as Protocol 3, follow-up session 1–3 weeks post-gameplay)  
**Dependency:** Requires GREEN gate from Protocol 3 (comprehension assessment)  
**When:** 1–3 weeks after Protocol 3 session; children do NOT replay L1–L6 in between

---

## 1. Objectives

**Primary:**
- Validate that children **retain fraction concepts 1–3 weeks after initial gameplay** (no drift to pre-test level)
- Measure **concept durability:** Can they still partition, identify, and compare without re-reading the game?
- Test **application to novel contexts:** Can they apply concepts to fractions they haven't explicitly practiced (e.g., 1/6, 1/8)?

**Secondary:**
- Quantify retention decay: Compare Protocol 3 post-test to Protocol 4 (1–3 weeks later)
- Identify **fragile knowledge:** If a child was Proficient in Protocol 3 but drops to Developing in Protocol 4, content may need spaced-repetition reinforcement
- Gather evidence that concepts are **stable enough for Grade 2 capstone (L7–L9)** without re-teaching L1–L6

**Decision gate (exit criteria):**
- **Green:** ≥70% retention of Protocol 3 post-test accuracy (i.e., if Protocol 3 = 80%, Protocol 4 ≥56%); no more than 1 misconception re-emergence per child
- **Yellow:** 50–69% retention; 1–2 misconceptions re-emerging; suggests need for spaced-repetition review or hint ladder scaffolding in L7–L9
- **Red:** <50% retention; 3+ misconceptions re-emerging or return to pre-test level; concept durability insufficient; recommend retention-loop in game or classroom review before L7–L9

---

## 2. Session Structure (30 min per child; follow-up only)

### Pre-Assessment Reminder (1 min)
1. **Greeting:** "Hi [name]! Remember the fraction game we played [duration ago]? Today I'm going to ask you some questions to see how much you remember."
2. **No review:** Do NOT show images or re-play levels
3. **Reassurance:** "Don't worry if you forgot — that's normal!"

### Retention Test (25 min)
- **Verbally-administered pencil-and-paper tasks** (same format as Protocol 3 post-test)
- **Novel fraction contexts** (fractions not explicitly in L1–L6, e.g., 1/6, 1/8, 2/4)
- **Comparison across denominators** (e.g., 1/2 vs. 1/4 vs. 1/8 on the same number line)

### Exit Interview (3 min)
- "Did you think about the game since we played?"
- "Do you remember anything about the game?"
- "If you saw the game again, do you think it would feel familiar?"

### Set B: Visual Transfer Tasks — RETENTION VERSIONS (paper, no screen)

**Task B1: Partition to sevenths (NEW denominator not in game)**
- **Prompt:** "Divide this rectangle into seven equal parts. Show where you'd cut. (Don't worry if it's not perfect.)"
- **Scoring:**
  - [ ] 7 equal parts (mastery transfer)
  - [ ] Attempted partition but unequal (procedural attempt; shows understanding of the task)
  - [ ] Random cuts or no attempt (no retention)
- **Follow-up:** "How did you know to make seven parts?"

**Task B2: Identify fractions on a new number line (0 to 1)**
- **Prompt:** "I'm going to mark positions on a number line. Tell me what fraction each is." (Mark four positions; child labels them.)
  - Position A: 1/8 (closer to 0)
  - Position B: 1/2 (middle)
  - Position C: 3/4 (closer to 1)
  - Position D: 5/8 (between 1/2 and 3/4)
- **Scoring per position:**
  - [ ] Exact (e.g., "1/2")
  - [ ] Directionally correct (e.g., "about half")
  - [ ] Incorrect or no answer
- **Avg accuracy:** ___/4 correct

**Task B3: Compare novel fractions (not explicitly in L1–L6)**
- **Prompt:** "Which is bigger: 1/6 or 1/3?" (Show on bar pictures)
- **Correct answer:** 1/3
- **Scoring:**
  - [ ] Correct (1/3)
  - [ ] Incorrect (says 1/6 or unsure)
- **Follow-up:** "How did you decide?"

## 4. Retention Decay Index (calculated post-session)

**Formula:**
```
Retention = (Protocol 4 Score - Protocol 3 Score) / Protocol 3 Score
```

**Interpretation:**
- **Retention ≥ 0.7:** Strong retention (decay <30%); child is stable
- **Retention 0.5–0.69:** Moderate retention (decay 30–50%); some drift, but not back to pre-test
- **Retention < 0.5:** Weak retention (decay >50%); concept fragile; needs spaced-repetition support

**Example:**
- Protocol 3 post-test: 8/10 (80%)
- Protocol 4 test: 6/10 (60%)
- Retention index: (60 − 80) / 80 = −0.25 (25% decay)
- **Interpretation:** Mild decay; child still at 60%, well above pre-test baseline (likely <30%); acceptable retention

## 5. Qualitative Retention Markers

**Ask during exit interview:**

**Q: "Did you think about the game since we played?"**
- [ ] Yes, frequently (good sign of engagement and concept consolidation)
- [ ] Once or twice (normal)
- [ ] No (not unusual; children forget quickly)

**Q: "Do you remember anything about the game? What was it?"**
- _Record verbatim:_
- [ ] Recalls specific task ("I remember cutting rectangles in half")
- [ ] Recalls general theme ("Something about cutting shapes")
- [ ] Can't recall anything

**Q: "If you saw the game again, do you think it would feel familiar?"**
- [ ] Yes, I'd remember how to do it
- [ ] Maybe, I'd figure it out quickly
- [ ] Probably not, it would feel new

## 9. Decision Logic (Gate Criteria)

**GREEN (full-scale L7–L9 expansion and validation):**
- Avg Protocol 3 → Protocol 4 retention ≥70%
- Avg retention index ≥0.70
- ≥75% of cohort show 0 misconception re-emergence
- ≥70% of cohort ready for L7–L9 without classroom review
- Avg concept decay ≤30%
- **Action:** Build L7–L9 content; conduct Round 3 validation with full curriculum

**YELLOW (L7–L9 expansion with spaced-repetition scaffolding):**
- Avg retention 50–69%
- Avg retention index 0.50–0.69
- 50–74% of cohort show 0 re-emergence
- 50–69% ready for L7–L9 without review; others need hint ladder or spaced-repetition challenge in L7
- Avg decay 30–50%
- **Action:** 
  - Build L7–L9 WITH spaced-repetition features (e.g., earlier denominators revisited as benchmarks in L8)
  - Add optional "Review Challenge" path (1–2 min per level) for children flagged as fragile in Protocol 4
  - Retest with 8–10 kids from this cohort on L7 after implementation

**RED (L7–L9 postponed; recommend classroom consolidation):**
- Avg retention <50%
- Avg retention index <0.50
- <50% of cohort show 0 re-emergence (>50% re-emerged misconceptions)
- <50% ready for L7–L9; majority need classroom review
- Avg decay >50%
- **Action:**
  - **Pause L7–L9 expansion.** Children need classroom review or practice (not game gameplay) to consolidate L1–L6
  - **Possible causes:** (a) Content was taught too fast (no time for consolidation between sessions), (b) Spacing was too close (played all levels in one day, no review period), (c) Framing or visual design made concepts feel disconnected
  - **Remediation:** Work with educators on spaced-repetition classroom activities (1–2 weeks post-game); add review loops to L1–L6 game; retest after classroom reinforcement

## 10. Interpreting Decay by Archetype

**If certain archetypes show weak retention, redesign them:**

| Archetype | Weak Retention? | Likely Cause | Recommendation |
|-----------|---|---|---|
| **Partition** | [ ] Yes  [ ] No | Procedurally learned, not conceptually | Add more visual reasoning (visual feedback of area) |
| **Identify** | [ ] Yes  [ ] No | Symbolic notation (1/2, 1/3) not reinforced | Add spaced-repetition symbolic challenge in later levels |
| **Equal-or-not** | [ ] Yes  [ ] No | Visual judgment fragile (confusing area vs. perimeter) | Strengthen area-checking visual in-game; add hint ladder |
| **Make** | [ ] Yes  [ ] No | Fold-and-shade not generalized to reasoning | Add compare-and-explain phase after make task |
| **Snap_match** | [ ] Yes  [ ] No | Isolated task (not connected to partition or compare) | Redesign to show connection: "This is the 1/2 you made earlier" |
| **Compare** | [ ] Yes  [ ] No | Relies on benchmark reasoning (fragile if not reinforced) | Add spaced-repetition benchmark challenge in L8 |

## 11. Facilitator Notes

1. **Timing matters:** Test at same time of day if possible (morning testing vs. afternoon fatigue can differ)
2. **Avoid leading:** "You said halves were equal before, are they still?" is leading; "Are these equal halves?" is neutral
3. **Decay is normal:** 20–30% decay over 2–3 weeks is expected; >50% is concerning
4. **Context cues:** If child improves on Protocol 4 (vs. Protocol 3), they may have had classroom review or family discussion — note it

---

## 12. Long-Term Follow-Up Recommendation

**Consider a 6-week follow-up** (Protocol 4b):
- Same cohort, same retention probes, 6 weeks after Protocol 3
- Measures whether decay continues or stabilizes
- If decay >50% at 6 weeks, escalate to spaced-repetition design for L7–L9
