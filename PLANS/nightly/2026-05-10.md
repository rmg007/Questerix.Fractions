## 🔴 Critical Findings

**[C-1]** New Pixi `EqualOrNotRenderer` registers no DOM mirrors in `A11yLayer` — Equal / Not-Equal choices are invisible to assistive tech (WCAG 4.1.2, 2.1.1) — *Source: a11y-auditor A11Y-001 → `src/interactions/pixi/renderers/EqualOrNotRenderer.tsx:165-174`* — *Why critical: this is the Phase-4 reference implementation; the pattern will propagate to the 10 Phase-6 skeleton renderers (`9a80527`) if not codified before they fill in.*

**[C-2]** Phase-1 spike Pixi buttons (`createButton`) are pointer-only — no keydown handler, no focus indicator, no DOM mirror, no `A11yLayer` registration — *Source: a11y-auditor A11Y-002 → `src/interactions/equal-or-not/renderer.ts:67-84`* — *Why critical: this renderer is wired into `LevelScreen.tsx:64`, so the live React app today exposes a question that cannot be answered with a keyboard or screen reader. Violates WCAG 2.1.1 Keyboard and 4.1.2 Name/Role/Value.*

**[C-3]** `TOUCH_TARGETS.smallButton = 40` violates the project ≥44×44 CSS-px touch-target rule — *Source: a11y-auditor A11Y-003 → `src/interactions/pixi/tokens.ts:75`* — *Why critical: every renderer that consumes this constant — including the 10 Phase-6 skeletons currently in the working tree — will ship sub-44 hit targets; locked C7/accessibility constraint.*

**[C-4]** `tweenAlpha` and `tweenPosition` in `pixi/visual.ts` ignore `prefers-reduced-motion` — exported as the canonical motion primitives for Pixi renderers — *Source: a11y-auditor A11Y-004 → `src/interactions/pixi/visual.ts:221-273`* — *Why critical: WCAG 2.3.3 violation that will inherit into every archetype Phase 6 fleshes out; the React-side `useTween` (`src/app/hooks/useTween.ts:19-24`) already shows the correct pattern — replicate it.*

**[W-2]** React shell screens ship unstyled HTML buttons — no min-size, no focus ring, no `aria-current` on active level — *Source: a11y-auditor A11Y-006 → `src/app/screens/MenuScreen.tsx:13`, `LevelMapScreen.tsx:15-22`, `LevelScreen.tsx:70`, `SettingsScreen.tsx:32`* — at 360 px viewport these will fail the 44×44 rule.
